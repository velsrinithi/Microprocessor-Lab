# Experiment 4: System Programs in 8086

##  Aim

To write 8086 assembly language programs using BIOS and DOS interrupts for display operations, string handling, and file management.

## Programs Included

### 1. Print Character 'I'

* Displays character 'I' on screen
* Uses BIOS interrupt (INT 10H)

### 2. String Comparison with Output

* Compares two strings
* Displays "EQUAL" or "NOT EQUAL"
* Uses DOS interrupt (INT 21H)

### 3. File Handling

* Creates a file
* Writes data to file
* Reads data from file
* Displays content on screen

### 4. Print Character 'T'

* Displays pattern of character 'T' on screen
* Uses cursor positioning and display interrupts

##  Concepts Used

* BIOS interrupts (INT 10H)
* DOS interrupts (INT 21H)
* File operations (create, read, write)
* String comparison
* Cursor positioning

##  Output

* Characters displayed on screen
* File created and content displayed
* Messages shown based on comparison result

##  Tools Used

* EMU8086 / 8086 Simulator
