# Array Operations in 8086

##  Aim

To implement array-based operations such as sorting, searching, and summation using 8086 assembly language.

##  Programs Included

### 1. Sorting

* Sorts an array of 16-bit numbers in ascending order
* Uses comparison and swapping technique

### 2. Searching

* Searches for a key element in an array
* Stores the result count

### 3. Sum of N Numbers

* Adds multiple 8-bit numbers
* Stores result as a 16-bit value

##  Concepts Used

* Array handling using SI register
* Looping using CX register
* Conditional jumps (JNZ, JBE)
* Memory addressing

##  Output

* Sorted array stored in memory
* Search result stored in RES
* Sum stored in SUM

##  Tools Used

* EMU8086 / 8086 Simulator
