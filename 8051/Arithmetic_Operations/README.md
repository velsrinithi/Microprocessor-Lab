# Experiment: 8051 Arithmetic Operations

## Aim
To perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division) using 8051 microcontroller.

---

##  Programs Included

### 1. Addition
- Adds two numbers stored in memory
- Stores result and carry

---

### 2. Subtraction
- Subtracts two numbers
- Stores result and borrow flag

---

### 3. Multiplication
- Multiplies two 8-bit numbers
- Stores result (low and high byte)

---

### 4. Division
- Divides two numbers
- Stores quotient and remainder

---

##  Concepts Used
- 8051 instruction set
- DPTR and external memory access (MOVX)
- Arithmetic operations (ADD, SUBB, MUL, DIV)
- Carry and borrow handling
- Register usage (A, B, R0–R2)

---

##  Memory Used
- Input data stored at: 9000H onwards
- Output stored in consecutive memory locations

---

##  Output
- Result of arithmetic operations stored in external memory

---

##  Tools Used
- Keil uVision / Proteus Simulator
- 8051 Microcontroller
